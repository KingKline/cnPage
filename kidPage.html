Yellow Belt / Level 4

Adventure: Creating with Repeat and For Element loops!

Requirements

Create a tilemap that includes at least 3 different types of tiles.
Use an image array and a loop to create multiple sprites, placing each on a different tile by using a variable that is changed inside the loop.
Hint

✅Use a [set list to] block to create a list of images.

✅Use a [for element] loop and a (value) bubble to make something happen to each element in an array. 

✅Use a [place sprite on top of] block to place sprites on the tilemap.

✅Use a [set [variable] to] block to set the position of the sprites.

✅Use a [change [variable] by] block to increase the column or row number each time the loop runs.

Use a loop to assign a sprite property for each value in an array of sprites.
Hint

✅Use a [for element] loop and a (value) bubble to make something happen to each element in an array.

✅Use an (array of sprites of kind) as the list in the [for element] loop.

Use sprite flags to apply a ghost or invisible effect to a sprite.
Hint

✅Use a [set sprite auto destroy] block and select a specific sprite flag from the dropdown menu.
